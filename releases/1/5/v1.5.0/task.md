# Release v1.5.0: Stream Architecture Refactoring

## Мета (Goal)
- Перенести логіку порядкового читання (для JSONL та CSV) на рівень `DBDriverProtocol` з `DB.js`.
- Гарантувати, що формат `.jsonl` та `.csv` безпечно ітерується рядок за рядком навіть якщо чанки розриваються посередині.
- Перевірити, що недоступні та неіснуючі файли викидають правильні помилки.
- Переконатися, що бінарні або не оброблювані файли (.bin) повертають оригінальний стрім.

## Acceptance Criteria (Definition of Done)
- [x] Логіка `stream` видалена з `DB.js` та перенесена в `DriverProtocol.js`.
- [x] Написано контрактні тести (`.spec.js`), що вкривають розрив чанків, правильне розділення та обробку помилок.
- [x] Всі тести проходять успішно (зелені).
